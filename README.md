# My PHP CMS

A custom PHP Content Management System (CMS) built from scratch as a personal learning project. This repository documents the development process, challenges, and improvements made while building a fully functional CMS.

## Project Status

**Stage One** – Initial release

During this stage I focused on building the core functionality of the CMS. Some UI improvements were implemented with the assistance of ChatGPT to enhance the appearance and usability of the public-facing dashboard.

## Features

- Content management
- Blog management
- User authentication
- User account management
- Media library and uploads
- Media settings
- Email system
- Password reset (Forgot Password)
- Registration settings
- Custom color picker
- Website templates

## Documentation

Additional documentation can be found in the `/docs` directory.

## Purpose

This project serves as both a learning exercise and a portfolio piece, demonstrating my experience building a CMS using PHP from the ground up while learning software development concepts and best practices.

"This project documents my journey of building a PHP CMS from scratch."
